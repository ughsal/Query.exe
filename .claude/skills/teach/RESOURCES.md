- [Node.js Official Documentation](https://nodejs.org/docs/latest/api/) — The authoritative source for Node.js APIs.
- [Express.js Guide](https://expressjs.com/en/guide/routing.html) — Official documentation on routing and middleware.
- [TypeScript Node.js Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) — Understanding how TS interacts with the Node runtime.

- [Node.js Best Practices GitHub](https://github.com/goldberli/node-best-practices) — A curated list of community-driven best practices.

- [Nodemon/Ts-node-dev](https://github.com/node-dev/ts-node-dev) — Essential for a productive development loop.
- [Winston/Pino](https://github.com/pinojs/pino) — Industry standard logging for production.
- [Zod](https://zod.dev/) — For environment variable and request body validation.
